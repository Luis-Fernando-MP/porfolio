&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 ¿Qué es&#63;
&lt;/h1&gt;
  
&lt;p&gt;
 Conjunto de conocimientos racionales y sistemático concentrados en el estudio de modelos abstractos &#40;Se enfoca en él ¿Qué hace&#63; Más que él ¿Cómo lo hace&#63;&#41; y los problemas que pueden solucionarse mediante algoritmos.
&lt;/p&gt;
  
&lt;p&gt;
 Aquí&#44; nos encontramos con el modelo abstracto más utilizado la &lt;b class=&quot;notion-text-bold&quot;&gt;&lt;a href=&quot;/1cf98397f7fa801abc01fdde64f16aea&quot; class=&quot;notion-text-href&quot;&gt;máquina de Turing&lt;/a&gt;&lt;/b&gt;&lt;b class=&quot;notion-text-bold&quot;&gt;&#44; &lt;/b&gt;un modelo razonable de la computación más potente posible&#44; y los modelos autómatas.
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Transpiladores
&lt;/h1&gt;
  
&lt;p&gt;
 Es un tipo de compilador de que traduce el código de un lenguaje de programación a otro. Se diferencia por transpirar archivos con código fuente. Un ejemplo perdurarte y popular es Babel que transpila el código de JavaScript moderno a la versión de JavaScript 2015 
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Interpretadores
&lt;/h1&gt;
  &lt;ul class=&quot;notion-bulleted&#95;list&quot;&gt;
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 Es un programa que ejecuta otros programas línea por línea&#44; sin compilar todo el código de antemano. Los intérpretes solo traducen lo necesario en tiempo real. 
&lt;/li&gt;
  
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 Esto permite que un mismo archivo fuente funcione igual en distintos sistemas &#40;por ejemplo&#44; una PC y una PS4&#41;&#44; siempre que tengan el mismo intérprete.
&lt;/li&gt;
  
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 Lenguajes como Python&#44; JavaScript&#44; Dart o MATLAB se interpretan; mientras que C o C&#43;&#43; necesitan compilación previa.
&lt;/li&gt;
  
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 Algunos lenguajes como Java o Python traducen el código a una forma intermedia &#40;.class&#44; .pyc&#41; y luego la interpretan. Esto mejora el rendimiento&#44; pero el acceso a variables es más lento porque todo se resuelve mientras el programa corre.
&lt;/li&gt;
  &lt;/ul&gt;
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Compiladores
&lt;/h1&gt;
  
&lt;p&gt;
 Es un programa que traduce todo el código fuente de un lenguaje de alto nivel a un lenguaje de bajo nivel como assembly o código máquina.
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Ensambladores
&lt;/h1&gt;
  
&lt;p&gt;
 Lenguaje de bajo nivel que permite escribir instrucciones directamente comprensibles por el hardware&#44; utilizando una sintaxis simbólica representativa del código máquina. Un programa ensamblador traduce estas instrucciones a lenguaje máquina&#44; permitiendo un control preciso sobre los registros y la memoria del procesador. Se utiliza principalmente en situaciones donde se necesita máxima eficiencia y control&#44; como en sistemas embebidos o desarrollo de drivers. 
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Lenguajes de Alto nivel
&lt;/h1&gt;
  
&lt;p&gt;
 Lenguajes de programación más cercanos al lenguaje humano&#44; diseñados para ser legibles y fáciles de escribir. Por ejemplo: Python&#44; Java&#44; JavaScript&#44; C#&#44; Ruby&#44; Swift.
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Lenguajes de Medio nivel
&lt;/h1&gt;
  
&lt;p&gt;
 Permite escribir programas estructurados&#44; con funciones&#44; control de flujo&#44; y código legible para humanos. Ofreciendo acceso directo a memoria &#40;punteros&#41;&#44; manipulación de bits y control de hardware&#44; similar al ensamblador. Por ejemplo: C&#44; C&#43;&#43;&#44; Rust&#44; Zig&#44; Go&#44; D
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Lenguajes de Bajo nivel
&lt;/h1&gt;
  
&lt;p&gt;
 Lenguajes más cercanos al lenguaje máquina&#44; con instrucciones específicas para el hardware. Por ejemplo: Assembly&#44; lenguaje máquina &#40;binario&#41;.
&lt;/p&gt;
  
&lt;h1 class=&quot;notion-heading&#95;1 notion-color-default&quot;&gt;
 Paradigmas de programación
&lt;/h1&gt;
  
&lt;p&gt;
 Son enfoques o estilos diferentes para resolver algoritmos&#44; definiendo cómo se estructuran y organizan los programas. Se podrían entender como valores&#44; prácticas&#44; guías o filosofías &#40;Un paradigma&#41; que dictan como podemos abordar la programación  con respecto a un problema en específico
&lt;/p&gt;
  
&lt;h2 class=&quot;notion-heading&#95;2 notion-color-default&quot;&gt;
 Imperativos &#40;Procedural&#41;
&lt;/h2&gt;
  &lt;ul class=&quot;notion-bulleted&#95;list&quot;&gt;
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 El código se organiza en una secuencia de instrucciones que la computadora ejecuta paso a paso. 
&lt;/li&gt;
  &lt;/ul&gt;
&lt;h2 class=&quot;notion-heading&#95;2 notion-color-default&quot;&gt;
 Declarativos
&lt;/h2&gt;
  &lt;ul class=&quot;notion-bulleted&#95;list&quot;&gt;
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 En lugar de especificar cómo se realizan las operaciones&#44; se define lo que se quiere lograr. La programación lógica y la programación de restricciones son ejemplos de paradigmas declarativos.
&lt;/li&gt;
  &lt;/ul&gt;
&lt;h2 class=&quot;notion-heading&#95;2 notion-color-default&quot;&gt;
 Funcionales
&lt;/h2&gt;
  &lt;ul class=&quot;notion-bulleted&#95;list&quot;&gt;
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 El código se organiza en funciones matemáticas que no tienen efectos secundarios&#44; fomentando la modularidad. Lenguajes como Haskell y Scala son ejemplos de lenguajes que apoyan fuertemente la programación funcional.
&lt;/li&gt;
  &lt;/ul&gt;
&lt;h2 class=&quot;notion-heading&#95;2 notion-color-default&quot;&gt;
 Orientados a objetos &#40;POO&#41;
&lt;/h2&gt;
  &lt;ul class=&quot;notion-bulleted&#95;list&quot;&gt;
&lt;li class=&quot;notion-bulleted&#95;list&#95;item notion-color-default&quot;&gt;
 El código se organiza en objetos que tienen datos &#40;atributos&#41; y comportamientos &#40;métodos&#41;. Los lenguajes como Java&#44; Python y C&#43;&#43; son ejemplos de lenguajes que soportan este paradigma.
&lt;/li&gt;
  &lt;/ul&gt;

<h1 class="notion-heading_1 notion-color-default">
 <b class="notion-text-bold">Historia</b>
</h1>
  
<p>
 La tesis no se impone&#44; se argumenta
</p>
  
<p>
 Plantear una postura firme y sustentada sobre un tema específico&#44; sujeta a evaluación por pares o jurados académicos. Su meta es convencer mediante la razón&#44; no la emoción.
</p>
  
<h3 class="notion-heading_3 notion-color-default">
 <b class="notion-text-bold">Una tesis científica debe:</b>
</h3>
  <ul class="notion-bulleted_list">
<li class="notion-bulleted_list_item notion-color-default">
 Presentar una <b class="notion-text-bold">conclusión clara</b> basada en pruebas.
</li>
  
<li class="notion-bulleted_list_item notion-color-default">
 Fundarse en datos objetivos y verificables.
</li>
  
<li class="notion-bulleted_list_item notion-color-default">
 Ser compatible con otras tesis válidas.
</li>
  
<li class="notion-bulleted_list_item notion-color-default">
 No presentar <b class="notion-text-bold">contradicciones lógicas</b>.
</li>
  
<li class="notion-bulleted_list_item notion-color-default">
 Basarse en <b class="notion-text-bold">hechos reproducibles</b>&#44; no creencias.
</li>
  
<li class="notion-bulleted_list_item notion-color-default">
 Evitar generalizaciones y ser <b class="notion-text-bold">específica</b>.
</li>
  
<li class="notion-bulleted_list_item notion-color-default">
 Hacer evidente el <b class="notion-text-bold">uso del método científico</b>.
</li>
  </ul>
<p>
 Podríamos enfocarnos en problemas <i class="notion-text-italic">vacíos de conocimiento o problemas sin resolver&#44; como:</i>
</p>

<tr class="notion-table_row">

<h3 class="notion-heading_3 notion-color-default">
 <b class="notion-text-bold">Estructura del proyecto de tesis</b>
</h3>
  <ol class="notion-numbered_list">
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Planteamiento del problema</b> → ¿Qué fenómeno vas a estudiar y por qué&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Justificación</b> → ¿Por qué es importante abordarlo&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Marco teórico</b> → ¿Qué dice la literatura al respecto&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Hipótesis</b> → ¿Qué crees que ocurrirá y por qué&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Objetivos</b> → ¿Qué te propones lograr&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Preguntas de investigación</b> → ¿Qué incógnitas vas a responder&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Delimitación</b> → ¿Hasta dónde abarca tu estudio&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Metodología</b> → ¿Qué enfoque y técnicas usarás&#63;
</li>
  
<li class="notion-numbered_list_item notion-color-default">
 <b class="notion-text-bold">Viabilidad</b> → ¿Con qué recursos&#44; en qué plazo y bajo qué condiciones&#63;
</li>
  </ol>

<h1 className="notion-heading_1 notion-color-default">
 Definición
</h1>
  
<p>
 La <b className="notion-text-bold">programación lógica</b> es un paradigma de programación que forma parte de la programación declarativa. Se fundamenta en la separación clara entre la lógica del problema y su mecanismo de resolución&#44; está expresada mediante la ecuación:
</p>
  
<blockquote className="notion-callout notion-color-brown_background">
 <div className="icon">
 null
 </div> 
 <div className="content">
Programa = Lógica &#43; Control
 </div>
</blockquote>
  
<h1 className="notion-heading_1 notion-color-default">
 Componentes Principales
</h1>
  
<table className="notion-table">
 
<tr className="notion-table_row">
 <td className="notion-table_cell"><b className="notion-text-bold">Componente</b></td><td className="notion-table_cell"><b className="notion-text-bold">Realiza</b></td><td className="notion-table_cell"><b className="notion-text-bold">Función</b></td><td className="notion-table_cell"><b className="notion-text-bold">Quién lo define</b></td>
</tr>
  
<tr className="notion-table_row">
 <td className="notion-table_cell"><b className="notion-text-bold">Lógica</b></td><td className="notion-table_cell">Determina las soluciones</td><td className="notion-table_cell">Define <b className="notion-text-bold">qué</b> problema resolver</td><td className="notion-table_cell">Programador: con hechos y reglas para representar la base de conocimiento</td>
</tr>
  
<tr className="notion-table_row">
 <td className="notion-table_cell"><b className="notion-text-bold">Control</b></td><td className="notion-table_cell">Ejecuta el proceso de búsqueda</td><td className="notion-table_cell">Define <b className="notion-text-bold">cómo</b> resolver el problema</td><td className="notion-table_cell">Intérprete: deducción lógica para dar respuestas</td>
</tr>
  
</table>
  
<h1 className="notion-heading_1 notion-color-default">
 Proceso
</h1>
  
<div className="notion-code">
 ```mermaid

  A1[Problema] --> A2[Análisis algorítmico]

 subgraph "Programación Lógica"
  B1[Problema] --> B2[Modelado del dominio]

  B4 --> B5[Resultado]
 end
```
 
</div>
  
<h1 className="notion-heading_1 notion-color-default">
 Principios Fundamentales
</h1>
  
<p>
 Se centra en <b className="notion-text-bold">declarar qué se sabe y qué se quiere saber&#44;</b> permitiendo que el sistema encuentre automáticamente las respuestas mediante razonamiento lógico. 
</p>
  
<p>
 Su filosofía central es:
</p>
  
<blockquote className="notion-quote notion-color-brown">
 <i className="notion-text-italic">Modelar problemas por medio de la abstracción&#44; utilizando un sistema de lógica formal que permite llegar a una conclusión por medio de hechos y reglas.</i>
</blockquote>
  
<h3 className="notion-heading_3 notion-color-default">
 Elementos Base
</h3>

<h2 className="notion-heading_2 notion-color-default">
 Ejemplo Práctico: Sistema de Diagnóstico Médico
</h2>
  
<p>
 Busquemos determinar posibles diagnósticos basándose en síntomas presentados por x pacientes.
</p>
  
<h3 className="notion-heading_3 notion-color-default">
 Base de Conocimiento
</h3>
  
<div className="notion-code">
 ```java
<b className="notion-text-bold">Hechos:
</b>
Haui -> presenta fiebre
Haui -> presenta dolor_garganta
Haui -> presenta ganglios_inflamados

Juli -> presenta tos_seca
Juli -> presenta fiebre
Juli -> presenta dolor_pecho
```
 
</div>
  
<h3 className="notion-heading_3 notion-color-default">
 <b className="notion-text-bold">Reglas de Diagnóstico:</b>
</h3>
  
<div className="notion-code">
 ```java
Si (X presenta fiebre) Y (X presenta dolor_garganta) Y 
	 (X presenta ganglios_inflamados) 
	 Entonces: X tiene posible_amigdalitis

Si (X presenta tos_seca) Y (X presenta fiebre) Y 
	 (X presenta dolor_pecho)
	 Entonces: X tiene posible_bronquitis
	 
Si (X posible_amigdalitis) Entonces: X requiere_antibiotico
Si (X posible_bronquitis) Entonces: X requiere_expectorante
```
 
</div>
  
<h3 className="notion-heading_3 notion-color-default">
 <b className="notion-text-bold">Consulta:</b>
</h3>
  
<div className="notion-code">
 ```java
¿Qué tratamiento requiere Haui?
```
 
</div>
  
<p>
 Las reglas médicas planteadas quedan de forma explícita y son fáciles de auditar&#44; modificar o expandir&#44; a diferencia de los algoritmos complejos de otros paradigmas de programación&#44; que seguramente aplicaríamos lógica dispersa para optimizar el código.
</p>

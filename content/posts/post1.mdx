## Introducción

JavaScript es un lenguaje de programación que se utiliza principalmente en el desarrollo web. Permite a los desarrolladores crear contenido dinámico y interactivo en las páginas web, mejorando la experiencia del usuario. Desde su creación, JavaScript ha evolucionado y se ha convertido en uno de los lenguajes más populares en el mundo del desarrollo.

## Características del Lenguaje

### Funcionalidades Principales

- Soporte para programación orientada a objetos
- Manipulación del DOM (Document Object Model)
- Soporte para eventos y manejo de interacciones del usuario
- Integración con APIs y servicios web
- Ejecución en el navegador y en el servidor (Node.js)
- Soporte para programación asíncrona

### Herramientas y Entornos

| Herramienta | Descripción                                                   |
| ----------- | ------------------------------------------------------------- |
| Node.js     | Entorno de ejecución para JavaScript en el servidor           |
| NPM         | Gestor de paquetes para JavaScript                            |
| Babel       | Transpilador para usar características modernas de JavaScript |
| Webpack     | Empaquetador de módulos para aplicaciones web                 |

## Instalación y Ejecución del Proyecto

### Requisitos Previos

1. Instalar [Node.js v20](https://nodejs.org/en/).
2. Instalar NPM (que viene con Node.js).

### Clonación del Repositorio

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Luis-Fernando-MP/javascript-intro.git
   cd javascript-intro
   ```

### Instalación de Dependencias

1. Instala las dependencias:

   ```bash
   npm install
   ```

### Ejecución del Proyecto

1. Ejecuta el proyecto en modo desarrollo:

   ```bash
   npm start
   ```

2. Para compilar el proyecto para producción:

   ```bash
   npm run build
   ```

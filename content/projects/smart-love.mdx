## Introducción

Smart Love Hotel es una plataforma moderna de reservas hoteleras ubicada en Lima, Perú. Desarrollada con tecnologías avanzadas, ofrece un sistema eficiente y seguro para la gestión de reservas y administración hotelera. El objetivo principal es proporcionar una experiencia de usuario óptima mediante una interfaz intuitiva y funcionalidades avanzadas.

## Características del Proyecto

### Funcionalidades Principales

- Reserva de habitaciones en tiempo real
- Calendario interactivo para selección de fechas
- Filtros avanzados de búsqueda
- Diseño responsive y moderno
- Modo oscuro y claro
- Autenticación de usuarios
- Gestión de reservas
- Panel de administración intuitivo
- Sincronización en tiempo real
- Análisis de disponibilidad

### Servicios Ofrecidos

| Servicio         | Descripción                          | Precio     |
| ---------------- | ------------------------------------ | ---------- |
| Habitación Doble | Habitación para dos personas         | $50/noche  |
| Suite            | Habitación con jacuzzi               | $100/noche |
| Desayuno         | Desayuno continental incluido        | $10        |
| Transporte       | Servicio de transporte al aeropuerto | $30        |

## Instalación y Ejecución del Proyecto

### Requisitos Previos

1. Instalar [Node.js v20](https://nodejs.org/en/).
2. Instalar PNPM globalmente:

   ```bash
   npm install -g pnpm
   ```

### Clonación del Repositorio

1. Clona el repositorio:

   ```bash
   git clone https://github.com/Luis-Fernando-MP/smartLove.git
   cd smart-love-hotel
   ```

### Instalación de Dependencias

1. Instala las dependencias:

   ```bash
   pnpm install
   ```

### Configuración de Variables de Entorno

1. Crea un archivo `.env` en la raíz del proyecto.
2. Copia las siguientes variables y configura sus valores:

   ```shell
   # API URL
   NEXT_PUBLIC_API_URL=

   # Clerk Authentication
   NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
   CLERK_SECRET_KEY=
   NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
   NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
   NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/
   NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/
   CLERK_WEBHOOK_USER_EVENTS=
   CLERK_WEBHOOK_USER_EVENTS_DEV=

   # Database
   DATABASE_URL=
   ```

### Ejecución del Proyecto

1. Ejecuta el proyecto en modo desarrollo:

   ```bash
   pnpm dev
   ```

2. Para compilar el proyecto para producción:

   ```bash
   pnpm build
   pnpm start
   ```
